# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of X-Ray Hand Tracker
- Real-time hand detection and skeletal visualization
- Support for tracking up to 2 hands simultaneously
- Color-coded output (blue for left hand, red for right hand)
- Real-time FPS counter and statistics display
- Responsive design for desktop and mobile devices
- Privacy-focused design with all processing done locally
- Comprehensive documentation and guides
- MIT License for open-source distribution

### Features
- 21 joint landmarks detection per hand
- Skeleton visualization with bone connections
- Live performance metrics (FPS, hand count, joint count)
- Camera permission handling
- Smooth real-time rendering using Canvas API
- MediaPipe integration for accurate hand detection

### Technical Details
- Built with vanilla JavaScript (no frameworks required)
- Single HTML file deployment
- Uses MediaPipe Hands model for detection
- WebRTC API for camera access
- Canvas API for visualization
